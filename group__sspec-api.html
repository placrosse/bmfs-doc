This function may fail if the storage specification is not able to be represented by a 64-bit unsigned integer.

Returns

int bmfs_sspec_parse ( struct bmfs_sspec *  sspec,
const char *  arg 
)

Parses a string describing a storage capacity.

A couple of examples of strings would be:

  • 3KB
  • 9B
  • 7 (equivalent to 7B)
  • 0
    Parameters
    sspecA storage specification structure. This parameter is initialized by the function.
    argA string representation of a storage capacity. The string must contain no spaces and is case-sensitive.
    Returns
    Zero on success, a negative error code on failure.
