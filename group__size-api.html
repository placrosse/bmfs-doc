Retrives the number of bytes of the storage specification.

This function may fail if the storage specification is not able to be represented by a 64-bit unsigned integer.

Parameters
sizeAn initialized storage specification.
bytesThe address of the variable that will receive the number of bytes of the storage specifier.
Returns
Zero on success, a negative error on failure.

◆ bmfs_size_parse()

int bmfs_size_parse ( struct BMFSSize *  size,
const char *  arg 
)

Parses a string describing a storage capacity.

A couple of examples of strings would be:

  • 3KB
  • 9B
  • 7 (equivalent to 7B)
  • 0
    Parameters
    sizeA storage specification structure. This parameter is initialized by the function.
    argA string representation of a storage capacity. The string must contain no spaces and is case-sensitive.
    Returns
    Zero on success, a negative error code on failure.
