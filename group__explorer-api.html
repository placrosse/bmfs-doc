◆ bmfs_explorer_init()

void bmfs_explorer_init ( struct BMFSExplorer *  explorer)

Initializes a BMFS file system explorer.

After calling this function, the disk callbacks should be set. Once the callbacks are set, then call bmfs_explorer_read and then the explorer will be ready for use.

Parameters
explorerAn uninitialized file system explorer structure.
